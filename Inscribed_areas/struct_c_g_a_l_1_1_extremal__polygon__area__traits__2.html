template<typename K>
struct CGAL::Extremal_polygon_area_traits_2< K >

This is an advanced class.

Advanced

The class Extremal_polygon_area_traits_2 provides the types and operations needed to compute a maximum area \( k\)-gon \( P_k\) that can be inscribed into a given convex polygon \( P\) using the function extremal_polygon_2.

AdaptableBinaryFunction class op: Point_2 \( \times\) Point_2 \( \rightarrow\) FT.

For a fixed Point_2 \( root\), op \( (p,\,q)\) returns twice the area of the triangle \( (root,\, q,\, p)\).
