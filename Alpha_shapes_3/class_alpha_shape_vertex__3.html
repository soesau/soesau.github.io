AlphaShapeVertex_3 Concept Reference
3D Alpha Shapes Reference » Concepts

Definition

The concept AlphaShapeVertex_3 describes the requirements for the base vertex of an alpha shape.

Refines
TriangulationVertexBase_3 if the underlying triangulation of the alpha shape is a Delaunay triangulation. RegularTriangulationVertexBase_3 if the underlying triangulation of the alpha shape is a regular triangulation. Periodic_3TriangulationDSVertexBase_3 if the underlying triangulation of the alpha shape is a periodic triangulation
Has models
CGAL::Alpha_shape_vertex_base_3 (templated with the appropriate triangulation vertex base class)
See also
CGAL::Alpha_status
